# 📈 Sales Forecasting Demand Intelligence System

A machine learning-based sales forecasting application that predicts future sales trends using historical data. The system helps businesses make data-driven decisions by providing accurate demand forecasts through an interactive web interface.

---

## 🚀 Features

- 📊 Sales forecasting using machine learning
- 📁 Upload custom sales datasets
- 🧹 Automatic data preprocessing
- 📈 Interactive data visualization
- 🤖 Future sales prediction
- 💻 User-friendly Streamlit interface
- 📉 Model performance evaluation

---

- **Python**
- **Streamlit**

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/RudrakshNautiyal/Sales-Forecasting-Demand-Intelligence-System.git
```

### 2. Navigate into the project

```bash
cd Sales-Forecasting-Demand-Intelligence-System
```

### 3. Create a virtual environment (Optional)

```bash
python -m venv venv
```

Activate it

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Streamlit application using:

```bash
streamlit run app.py
```

The application will be available at

```
http://localhost:8501
```
